## <samp>whoami</samp>

Software engineer based in **Tokyo** 🗼 — I build web apps, APIs, and the occasional
multiplayer game. Two-time **Global Game Jam** survivor. Dig through my repos and you'll
find browser games, a from-scratch RSA in C, a plain-text Pokédex API in Go, a catalog of
every public Age of Empires II data API, and the Astro site behind
[usta.dev](https://usta.dev). I care about the boring parts: types that hold, endpoints
that don't lie, and interfaces that get out of the way.
